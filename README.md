# Recipe Generation Website

An AI-powered recipe generation website built with Python Flask. Input ingredients and get personalized recipe suggestions with step-by-step cooking instructions.

## Features

- **Ingredient-based Search** - Enter available ingredients and get matching recipes
- **Recipe Suggestions** - AI-powered recipe generation
- **Step-by-step Instructions** - Detailed cooking instructions
- **Clean UI** - Simple and responsive web interface
- **Backend API** - Flask-based REST API for recipe processing

## Tech Stack

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript
- **AI/ML**: Python libraries for recipe processing

## Project Structure

```
.
├── app.py                 # Main Flask application
├── index.html             # Main web page
├── result.html            # Recipe results page
├── style.css              # Custom styles
└── README.md              # Project documentation
```

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
# Clone the repository
git clone https://github.com/niha1905/recipe-generation-website.git
cd recipe-generation-website

# Install dependencies
pip install -r requirements.txt  # if requirements.txt exists
```

### Running the App

```bash
python app.py
```

Open your browser and navigate to `http://localhost:5000`
